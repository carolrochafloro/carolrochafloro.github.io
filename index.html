Form - controle de mensagens

Aplicação desenvolvida em ASP.NET 8 utilizando o padrão MVC. Ao se cadastrar e logar no sistema, o usuário pode receber uma URL personalizada que direciona para uma página de contato. A mensagem será enviada por e-mail através do Sendgrid.

A solução contém dois projetos: um MVC e um worker. Os dados do contato são enviados para o RabbitMQ e o worker consome a mensagem, realiza o envio e atualiza o banco de dados conforme o status de envio do e-mail.

A aplicação consome a API do Bitly para gerar uma URL encurtada.
