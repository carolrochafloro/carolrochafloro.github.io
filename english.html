Form - e-mail management

MVC app developed in ASP.NET 8. After the user registers and logs in, they can receive a personalized URL Aplicação that directs to a contact page. The message sent on that page will be emailed using Sendgrid.

The solution contains two projects: an MVC app and a worker. The contact data is sent to RabbitMQ and the worker consumes the message and emails it. Than, it updates the database according to the status of the message.

The app consumes Bitly API to shorten the URL.

Task Timer

PostgreSQL was used as the database with EF Core as the ORM, and Microsoft Identity to manage user authentication.
